# 03 - Recommended Project Structure (Non-Mandatory)

## Goal
Provide an optional structure for medium-size apps without enforcing it.

## Suggested Layout
```
app/
  Controllers/
  Services/
  Middleware/
routes/
config/
var/
public/
```

## Deliverables
- Documentation section describing the optional structure.
- Example skeleton (no code changes required).

## Acceptance Criteria
- No runtime enforcement; documentation only.
- Works with existing `loadApp()` and config loading.

## Notes
- Keep it light and optional.
